Librería básica punto de partida para otras librerías

## Instalación

  `yarn add general-library-starter`

## Uso

Esta librería sirve como ejemplo base para generar otras librerías de código
abierto publicadas en GitHub.

## Test

  `yarn test`

## Cobertura

  `yarn cover`
